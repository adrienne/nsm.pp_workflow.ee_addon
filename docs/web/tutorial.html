Getting started

Before you can use NSM Publish Plus you must configure the module using the Extension Settings.

Assigning an entry state and review period

Now that your extension settings are configured and the module ready for use, you can start setting assigning entry states and review dates.

Create a new channel entry or edit an existing entry and open the 'Workflow' tab. The tab is shown in two sections, 'Entry state' and 'Next review date'. Once you have set the options in the Workflow form you can continue with the content submission.

Entry state

The entry state refers to the status of the content in Workflow, not the ExpressionEngine status, and is set as one of the following:

  • Pending - Item is awaiting approval
  • Approved - Item is approved for the website
  • Review - The entry is deemed old and requires review for content relevancy

When entries are reviewed by the module and a state of 'Pending' or 'Approved' is found and the entry is due for review the workflow state will be set to 'Review' and an email notification will be sent to the nominated addresses.

Next review date

Website administrators can specify the next review date in this part of the form. For new entries an estimated review date will be suggested based on the extension settings defined for the channel. Alternatively the administrator can specify their own date and time by using the 'New review date' radio button and input field.

Automating entry reviews

The URL shown in the automation section of the extension settings is the URL that will need to be added to a server CRON job. Visiting / pinging this URL will search for channel entries that are due for review within the enabled channels.

The automation URL will look something like: http://yoursite.com/?ACT=24

Each server setup has a unique CRON implementation. Ask your web host for more information or try Google.

Reviewing entries in the dashboard

When an entry is due for review it will appear in the NSM Publish Plus Workflow module dashboard. The dashboard is an easy place to find these entries. The information displayed includes the entry id, the entry name, a link to the edit form for the entry, the channel name, status, state, last edit date and review date.

Click on the title of a channel entry to start editing the content. When you have finished updating your content in the Publish tab open the 'Workflow' tab and update the workflow state and set a new review date. You will most likely choose to use the 'Approved' state and 'Estimated review date from now' (second option) for Next review date. After you have submitted the content with a state other than 'Review' that entry will no longer appear in the module dashboard until it is due for the next review.
